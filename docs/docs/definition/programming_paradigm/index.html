编程范式

定义#

什么是编程范式？#

编程范式(programming paradigm)是一种根据编程语言的特性对编程语言进行分类的方法。语言可以分为多种范式。

一些范式主要关注对语言执行模型的影响，例如允许副作用，或者操作序列是否由执行模型定义。其他范式主要关注代码的组织方式，例如将代码与代码修改的状态一起分组。还有一些人主要关注句法和语法的风格。

常见的编程范式包括：

  • 命令式(imperative)：程序员指示机器如何改变其状态
    • 面向过程(procedural)：将指令分组为程序(procedure)
    • 面向对象(object-oriented)：将指令与它们操作的状态部分组合在一起
  • 声明式，其中程序员只声明所需结果的属性，而不是如何计算它
    • 函数式(functional)：期望的结果，被声明为一系列函数应用程序的值
    • 逻辑(logic)：期望的结果，被声明为关于事实和规则系统的问题的答案
    • 数学(mathematical)：期望的结果，被声明为优化问题的解决方案
    • 反应式/响应式(reactive)：使用数据流和更改的传播，声明所需的结果
