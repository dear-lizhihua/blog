第一个参数：application 名称#

第二个参数：application 对象#

传递一个 async 函数，这个 async 函数就是代表了应用程序的包裹函数。

为什么要通过 async (props) => {} 包裹 bootstrap, mount, unmount？

这是为了向 application 传递 props，所以在外面额外包裹一个函数，用于透传参数。

第二个参数返回值：application 生命周期#
